# Deployment & Environment variables\nSet the following in Netlify UI (Site settings → Build & deploy → Environment):\n\n- TELEGRAM_BOT_TOKEN = <bot-token-from-BotFather>\n- TELEGRAM_CHAT_ID = <numeric-chat-or-group-id>\n- NEXT_PUBLIC_API_BASE = <optional public API base>\n\nSecurity\n- Keep tokens secret (Netlify env is safe).\n- Do not commit secrets to the repo.\n- This project sends applicant data and attached documents directly to Telegram for HR review; ensure your Telegram chat/group is privately configured.\n\nTesting locally\n- Install Netlify CLI: npm i -g netlify-cli\n- netlify dev (reads netlify.toml and runs functions locally)\n\nNotes\n- This scaffold sends Telegram messages and uploads documents to Telegram. For production, ensure you comply with privacy rules and have HR consent.\n
